🧪 Prompt2Test — Project Overview

UI Screens — Built & Deployed

Agent

AI Test Plan Generation
QA types a natural language prompt. Claude Sonnet 4.5 generates a structured plan with actions and expected outcomes.
✅ Complete

Live Browser Automation
Playwright MCP executes test steps in a real Chromium browser. ECS Fargate task spins up on demand, stopped when done.

Watch the browser execute in real-time via embedded noVNC iframe. Pop-out button opens a full window.

Test Replay
Re-execute a saved test case from the Inventory without re-generating the plan. Uses the stored replay script.

Test Case Inventory
Save, edit, delete test cases. Filter by environment and service. View last run result per test case.
✅ Complete

Semantic Search

Key Architecture Decisions
DecisionChoiceWhy
Config storageDynamoDBBrowser SDK calls DynamoDB directly via Cognito IAM — no server proxy needed. SSM cannot be called from the browser.
Team membershipCognito GroupsTeam resolved from JWT cognito:groups claim at login — zero extra lookups.
Browser sessionsECS Fargate on-demandOne task per test run, auto-stopped on completion. Zero idle cost — pay only when tests run.
AI agent runtimeBedrock AgentCoreManaged container runtime — no EC2/ECS service to maintain. Scales automatically.
Test case storageAurora + pgvectorVector embeddings enable semantic duplicate detection and similarity search.
InfrastructureAWS CDK (TypeScript)Full IaC — entire platform deployable to a new AWS account with one command.
