Smart Replay Cost — LLM-Assisted Retry on Failure

Deterministic replay costs $0 LLM tokens when all steps pass. Smart retry only calls the LLM when a step fails. The LLM adapts HOW to interact (wait, new selector) but never changes WHAT is tested.

How it works: Each test case runs as an agent loop — multiple LLM calls that accumulate context. The total input tokens is the sum across all calls, which can approach the context window limit for complex tests. Output tokens are always much smaller. You pay only for tokens consumed, not time.
