How to Reduce Costs

1. Use cheaper models for simple tests — Nova Lite for smoke tests, Haiku for retries.
2. Batch API (50% off) — for non-urgent suite runs, use Bedrock batch processing.
3. Prompt caching — system prompt + plan steps can be cached (90% cheaper on re-reads).
4. Minimize DOM snapshots — only snapshot when verifying, not on every click.
5. Headless replay mode — deterministic replay uses zero LLM tokens (just Playwright).

Important: Replay Mode = $0 LLM Cost

Once a test is automated and the replay script is saved, replaying it in QA/UAT/PROD costs zero LLM tokens. Only the first automation in DEV uses the LLM. Promoted tests run as deterministic Playwright replay — no AI involved.
