Replay Mode = $0 LLM Cost

Once a test is automated and the replay script is saved, replaying it costs zero LLM tokens. Only the first automation uses the LLM. Promoted tests run as deterministic Playwright replay — no AI.

How to Reduce Costs

1. Cheaper models for simple tests — Nova Lite for smoke, Haiku for retries.
2. Batch API = 50% off — for non-urgent suite runs.
3. Prompt caching = 90% off — cached system prompts on repeat calls.
4. Headless replay = $0 — deterministic replay uses zero LLM tokens.
